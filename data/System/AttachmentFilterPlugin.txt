---++ Configuration
=AttachmentFilterPlugin= preferences are set in [[%SCRIPTURLPATH{"configure"}%]].
%BR% The configuration is split into two preferences:
   * =$Foswiki::cfg{Plugins}{AttachmentFilterPlugin}{Behaviour}=
      * Defines the action if an attachment of a type listed in =FiletypeFilter=
      is going to be upload. Defaults to =block=
   * =$Foswiki::cfg{Plugins}{AttachmentFilterPlugin}{FiletypeFilter}=
      * Semicolon seperated list of file extensions for which =AttachmentFilterPlugin=
      shall execute its defined ={Behaviour}=. Defaults to =''=.

_Example:_
<verbatim>
$Foswiki::cfg{Plugins}{AttachmentFilterPlugin}{Behaviour} = 'block';
$Foswiki::cfg{Plugins}{AttachmentFilterPlugin}{FiletypeFilter} = 'bmp;exe;com;zip;tar';
</verbatim>

*Note* %BR%
Setting preference ={Behaviour}= to =allow= while not explicitly telling
=AttachmentFilterPlugin= which file types you want to allow (={FiletypeFilter}=),
all incoming file uploads will be declined!
